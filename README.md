# og

usage:

```og.py https://blog.kissmetrics.com/open-graph-meta-tags/```

  "twitter_title": "What You Need to Know About Open Graph Meta Tags for Total Facebook and Twitter Mastery",
  "og_image_height": "200",
  "og_image_width": "850",
  "og_image_type": "image/jpeg",

}
